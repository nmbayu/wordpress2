]]> Ventoy adalah aplikasi booting Sistem Operasi lewat flashdisk yang sangat praktis. Kita tak perlu memformat flashdisk kita setiap kita akan memasukan ISO OS atau Distro, untuk keperluan live booting dari flashdisk, dengannya kita hanya perlu sekali format, dan setelah itu kita bisa langsung menambahkan ISO pada flasdisk kita. ISO tersebut bukan hanya dari satu Sistem Operasi, tapi bisa kita tambah berapa pun ISO sistem operasi yang kita mau, tergantung dari kapasitas flashdisk yang kita gunakan, serta kita dapat menghapus salah satu ISO yang sudah tidak kita gunakan lagi. Dan satu lagi Ventoy adalah aplikasi Open Source, jadi gratis untuk kita gunakan. Serta Ventoy sudah support UEFI atau Legacy,dan partisi MBR atau GPT. Dan juga Ventoy support penginstalan Sistem Operasi ke PC atau laptop kita. Ventoy bisa di unduh di link ini : https://www.ventoy.net/en/download.html

Adapun cara menggunakannya adalah berikut ini:

1.  Windows

Di Windows kita cukup dengan hanya mendownload aplikasi Ventoy yang berbasis Windows, ekstrak terus klik file exe nya, Ventoy akan jalan automatis serta mendetek flashdisk kita. Lalu kita tinggal klik install, dan setelah itu taruh file-file ISO yang mau kita gunakan ke flashdisk.

2. Linux

Untuk di linux, cara menggunakannya berbasis teks, yang dijalankan melalui terminal.

Caranya cari dimana flashdisk kamu berada dengan perintah :

$ sudo fdisk -l

nanti keluar info flashdisk kita. contoh :

Units: sectors of 1 * 512 = 512 bytes
Sector size (logical/physical): 512 bytes / 512 bytes

Bisa dilihat disitu bahwa flashdisk berada di /dev/sdb

Setelah di ketahui yang mana flashdisk kita maka kita, kita masuk ke folder ventoy yang telah kita download dan ekstrak, di dalam folder tersebut kita klik kanan, terus open terminal here.

Terus ketik perintah :

$ sudo sh Ventoy2Disk.sh -I /dev/sdb

akan keluar seperti contoh dibawah ini:

]]> 1. Steam OS

Mari kita mulai  dengan Steam OS. Ini adalah salah satu distro game terpopuler saat ini. Ini karena ini dikembangkan dan di maintenant oleh Valve.

Berbasis pada OS Debian 8 dan menggunakan lingkungan desktop GNOME. Untuk membuatnya nyaman bagi para gamer, ini mendukung berbagai jenis joystick, gamepad, dan kartu grafis setelah pemasangan. Anda dapat melanjutkan dan menginstal driver untuk perangkat keras Anda secara manual jika tidak didukung.

Menginstal Steam OS sudah diinstal sebelumnya dengan Steam dan banyak game dari Steam Store. Semuanya dirancang dan disempurnakan dengan mempertimbangkan game, jadi Anda tidak perlu menginstal apa pun untuk bermain game di Steam.

Sayangnya, ini tidak disarankan jika Anda memiliki PC lama karena persyaratan perangkat keras yang tinggi. Namun bagi penggemar game, hal tersebut lumrah, mengingat game – game masa kini yang memang membutuhkan hardware yang lebih baru dan spesifikasi yang tinggi.

2. Ubuntu GamePack

Distro Linux lain adalah Ubuntu GamePack. Ini bukan distro Ubuntu biasa tetapi lebih berbasis pada Ubuntu.

Hal tentang Ubuntu GamePack tidak memiliki game yang diinstal sebelumnya seperti Steam OS. Namun, ini sudah diinstal sebelumnya dengan klien Steam, Wine, PlayOnLinux, dan Lutris. Ini seperti persimpangan jalan tempat game di Linux, Windows, Konsol, dan Steam bisa dimainkan.

Ini membuatnya menjadi OS hybrid dalam arti tertentu, terutama karena ini juga mendukung Oracle Java dan Adobe Flash. Ini berarti bermain game online juga dimungkinkan. Faktanya, ada lebih dari 6 ribu game Linux dan Windows yang dijamin akan diluncurkan di Ubuntu GamePack.

Jika Anda pernah mencoba Ubuntu sebelumnya dan merasa nyaman dengannya, gunakan distro ini.

3. Fedora Games Spin

Fedora Games Spin  distro dirancang untuk para gamer. Fakta bahwa ada ribuan game yang sudah diinstal didalam distro ini dengan tools untuk menginstal game yang lebih banyak membuatnya layak untuk di coba.

Game yang sudah diinstal di distro berkisar dari First Person, Real Time Strategy, dan banyak lagi. Semuanya tersedia untuk menunjang kegiatan bergame anda.

Tidak seperti kebanyakan distro Linux, ini cepat, ringan, dan stabil.

Jika anda masih ingin memiliki game PlayOnLinux, Wine, atau Steam, Anda harus menginstalnya secara manual karena tidak diinstal sebelumnya secara default. Selain itu, Anda juga perlu menginstal driver baru karena tidak disertakan dengan instalasi.

Jika Anda mencari distro Linux game dan merasa nyaman dengan lingkungan desktop XFCE, ini cocok untuk Anda.

4. SparkyLinux – Gameover Edition

Distro ini sudah memiliki banyak sekali game pra-instal (sudah terintall di system) yang dapat Anda mainkan, tetapi program APTus Gamer menawarkan Anda lebih banyak opsi untuk bermain. Ini berisi daftar emulator yang dapat Anda instal di sistem Anda. Namun, jika Anda ingin bermain game di Steam, PlayOnLinux, atau Wine, Anda harus menginstalnya secara manual karena game-game tersebut belum terpasang sebelumnya.

Secara keseluruhan, jika Anda adalah seorang gamer yang rajin bermain game dengan emulator, Anda bisa menginstal SparkyLinux – edisi Gameover.

5. Manjaro Gaming Edition

Sebenarnya Manjaro Gaming Edition ini adalah Distro unofficial (tidak resmi) berbasis Manjaro, makanya tidak ditemukan di website resmi Manjaro. Manjaro Gaming Edition didedikasikan untuk para gamer yang cenderung membuat konten. Kami dapat mengatakan bahwa ini untuk para gamer yang menyukai pembuatan konten karena ini sudah diinstal dengan tools untuk membuat konten.

Beberapa di antaranya termasuk Audacity, aplikasi streaming, alat tangkapan layar, perangkat lunak chatting, dan banyak lagi.

Salah satu hal yang menjadikan Manjaro OS yang harus Anda pertimbangkan adalah kenyataan bahwa ini sudah diinstal dengan driver kartu grafis populer. Mereka tidak hanya diperbarui secara berkala tetapi dapat dioptimalkan untuk pengalaman bermain game yang lebih baik.

Ini juga sudah diinstal dengan emulator seperti Dolphin, DOSbox, FCEUX, Kega Fusion, PCSX, PCSX2, PPSSPP, RetroArch, Stella, VBA-M, Yabause, dan ZSNES. Jika Anda ingin menikmati pengalaman bermain game Anda di emulator ini, Manjaro Gaming Edition akan sempurna.

Satu-satunya masalah dengan distro ini adalah kenyataan bahwa ini adalah distro berbasis Linux Arch. Jika Anda ingin bermain game uap, Anda harus menginstal klien Steam secara manual terlebih dahulu. Tetapi karena klien Steam tidak didukung oleh Arch, akan ada masalah di sana-sini.

Secara umum, gamer yang menyukai kreasi konten kemungkinan besar akan cocok dengan anda.

Memilih distro Linux terbaik untuk bermain game mungkin tampak menantang, tetapi memeriksa ulasan seperti ini akan memberi anda pemahaman yang lebih dalam tentang distro mana yang cocok untuk kebutuhan game anda.

]]> Sering kali kita dihadapkan pada masalah tidak bisa terkoneksinya jaringan kita dengan laptop/ PC kita yang bersistem operasi Windows 10 (can’t connect to this network). Meskipun Windows 10 adalah sistem operasi terbaru, belum tentu terbebas dari masalah.

Dibawah ini tercantum beberapa solusi untuk masalah – masalah tersebut :

1. Reinstall Network Adapter

Cara pertama yaitu reinstall network adapter, karena bisa saja hal yang menyebabkan “can’t connect to this network” ialah karena masalah software di network adapter kita, oleh karena itu kita uninstall dulu network adapternya baru di install lagi.

Adapun langkah – langkah reinstallnya ialah sebagai berikut :

  1. Tekan Windows key + X.
  2. Pilih Device Manager dari daftar device.
  3. Didalam Device Manager setting, beralih ke network adapter.
  4. Klik kanan di network adapter setting
  5. Klik Uninstall, dan klik OK.
  6. Ketika di uninstall maka akan automatis restart dan menginstal ulang dengan sendirinya.

2. Disable/ Non Aktifkan IPv6

Error jaringan pada Windows 10, bisa terjadi berkali – kali ialah bisa disebabkan miskonfigurasi di dalam daftar IP Network Protokol setting anda. Anda bisa mendisable/ menonaktifkan IPv6 Protokol Network dan kemudian bisa memecahkan masalahnya dengan beberapa langkah berikut :

  1. Buka Network and Sharing Center.
  2. Cari Connection Section dan klik koneksi sekarang.
  3. Koneksi status window akan terbuka.
  4. Klik Properties Button.
  5. Cari Internet Protocol Version 6 (TCP/IPv6).
  6. Hapus centangnya, dan sekarang IPv6 sudah dinonaktifkan.
  7. Tekan enter dan restart komputer.

3. Update Driver Network Adapter

Solusi selanjutnya ialah update network adapter driver, kemungkinan besar terjadi konflik atau bug dalam device network adapter anda sehingga koneksi pada jaringan terganggu. adapun langkah – langkahnya :

  1. Buka Windows + X.
  2. Pilih Device Manager.
  3. Pilih Network Adapter dan cari Jenis dan Tipe Network Adapter anda.
  4. Lalu pilih update.

4. Forget atau Lupakan Koneksi Wifi Anda

Salah satu solusi lain ialah memforget/melupakan jaringan Wifi anda, di mana langkah – langkahnya sebagai berikut :

  1. Open Settings.
  2. Pergi ke Network dan Internet.
  3. Pergi ke bagian Wifi
  4. Klik Manage Wifi settings.
  5. Pergi ke Manage Known Network.
  6. Kemudian pilih jaringan hotspot yang tidak bisa tersambung dan klik forget.

5. Disable dan Re-enable Wireless Adapter

Solusi yang terakhir adalah dengan mendisable dan reenable kembali Wireless adapter anda. Adapun langkah – langkahnya adalah :

  1. Buka Control Panel.
  2. Klik Network and Internet.
  3. Pilih Network Connection yang akan di disable dan reenable kembali.
  4. Klik kanan pilih disable, terus restart
  5. Setelah Restart maka akan automatis terenable kembali.

Demikianlah solusi untuk masalah Windows 10 “can’t connect to this network” semoga bermanfaat.

]]> Beberapa hal dapat membuat laptop atau komputer Anda menjadi lebih buruk. Hal-hal yang mungkin hanya membutuhkan waktu sehari dalam pengerjaannya dapat mengakibatkan frustrasi berminggu-minggu untuk menyelesaikan suatu pekerjaan. Ini karena sistem operasi akan selalu berubah untuk semakin mengoptimalkan softwarenya maupun perangkatnya. Jika hal ini tidak berjalan dengan baik, hal tersebut dapat membuat komputer tidak berjalan sebagaimana mestinya.

Sangat penting untuk menjaga komputer tetap dalam kondisi prima. Ada kebutuhan untuk pemeliharaan perangkat secara reguler demi kelancaran pengoperasian, sehingga gangguan yang terjadi seperti virus dan lainnya dapat dihindarkan.

Berikut adalah tips untuk menjaga laptop atau komputer Anda berjalan secara optimal:

Tips untuk perangkat lunak perawatan komputer meliputi:

1.  Periksa secara teratur untuk pembaruan dari produsen dan pastikan pemasangan yang benar. Jika Anda membiarkannya diabaikan, komputer Anda tersedia untuk peretasan, terutama karena terhubung online. Atur jadwal bulanan atau mingguan untuk memastikan sistem operasi terbaru.
Menjalankan laporan sistem jaringan untuk mengidentifikasi data apa pun yang aktivitasnya tidak normal.

2. Otomatiskan laporan sistem sehingga proses pencadangan terjadi setiap hari secara automatis, serta akan memberi tahu Anda tentang adanya pembaruan yang terbaru.

3. Perbarui kebijakan IT di kantor karena teknologi akan terus berubah dan selalu ingat untuk mengganti atau memperbarui perangkat lunak secara tepat waktu. Tujuan memperbarui adalah untuk menghindarkan para peretas maupun virus yang menyebabkan komputer bermasalah, jika software tidak diperbarui maka terbuka kemungkinan untuk peretas maupun virus untuk mengambil keuntungan. Perbarui definisi firewall dan virus agar berfungsi dengan baik. Gunakan penyimpanan cloud untuk pemulihan dan pelestarian data secara teratur.

Tip untuk pemeliharaan komputer untuk perangkat keras meliputi:

1. Berikan pembersihan rutin dengan membersihkan keyboard. Jauhkan kotoran dan debu agar unit daya dan kipas angin tidak macet. Gunakan vacum cleaner ukuran kecil (biasa buat mobil) untuk membersihkan debu.

2. Pastikan perangkat keras selalu upto date. Perlambatan jaringan dapat terjadi seiring waktu penggunaan peralatan yang sudah ketinggalan zaman.

3. Selalu simpan perangkat keras Anda di tempat yang kering dan aman untuk mencegah kehilangan data atau kerusakan bencana seperti kerusakan atau banjir.

4. Jaga ruang kosong pada hardisk yang cukup. Ini karena jika ruang hardisk penuh, kinerjanya akan lambat atau mungkin terhenti/hang, dan dapat menjadikan sangat mengganggu bagi pengguna.  Akan lebih baik  pindahkan data tersebut ke penyimpanan cloud, sehingga memberikan ruang kepada hardisk.

5. Putuskan sambungan perangkat keras yang tidak diperlukan dan pastikan firewall yang diperlukan untuk melindungi data aktif untuk akses tanpa hambatan ke jaringan yang terbuka.

Dengan adanya pemeliharaan pada komputer maupun laptop, maka diharapkan terjadinya peningkatan kinerja atau performa dari device tersebut, sehingga berimbas pada kelancaran operasi sehari – hari pada perusahaan.

]]> 1. Gunakan Verifikasi Dua Langkah untuk Email

Hal nomor satu yang perlu Anda lakukan untuk melindungi diri Anda secara online adalah mengamankan alamat email Anda. Peretas yang memiliki akses ke email Anda dapat mengubah akun lain.

Jika memungkinkan, Anda perlu menyiapkan verifikasi dua langkah untuk email Anda.

Anda harus menggunakan kata sandi dan metode verifikasi lain sebelum mendapatkan akses. Ini adalah langkah efektif yang akan mencegah peretasan.

2. Buat Kata Sandi Unik

Langkah lain dalam menghentikan peretas adalah membuat kata sandi yang baik. Mayoritas pengguna internet membuat kata sandi yang sangat buruk. Pada tahun 2014, misalnya, lima kata sandi paling populer adalah:

123456
kata sandi
12345
12345678
qwerty

Penjahat dunia maya yang memecahkan kata sandi mengunggah informasi ke basis data online. Itu berarti kapan saja Anda menggunakan kata sandi pada basis data, kata sandi itu telah terpapar ke peretas.

3. Perbarui Sistem Operasi Komputer atau Perangkat Lunak Anda

Kiat lain tentang cara mencegah peretas adalah tetap mengikuti pembaruan perangkat lunak. Pembaruan penting ini sering memiliki tambalan yang memperbaiki kerentanan.

Gagal memperbarui dapat membuat Anda terbuka terhadap serangan. Yang terbaik adalah mengatur ponsel atau komputer Anda untuk mengunduh pembaruan secara otomatis begitu tersedia.

4. Lindungi Koneksi Internet Anda dengan Kata Sandi

Terlalu banyak pengguna internet gagal memproteksi koneksi internet mereka sendiri.

Tanpa kata sandi, semua data yang Anda kirim dan terima benar-benar terbuka. Itu berarti siapa pun yang berada dalam jangkauan router Anda bisa mendapatkan data ini.

Penjahat juga dapat menggunakan jaringan terbuka untuk menyebarkan infeksi atau mencuri bandwidth Anda. Paling buruk, mereka bisa menggunakan koneksi Anda untuk melakukan serangan DDoS ilegal.

5. Mempertimbangkan kembali Menghubungkan ke Jaringan WiFi Terbuka

Memahami bahwa Anda menghadapi risiko paparan yang sama ketika Anda terhubung ke jaringan WiFi publik yang terbuka.

Berikut adalah beberapa tips tentang cara melindungi diri Anda dari peretas di jaringan publik:

Selalu matikan pengaturan ‘koneksi otomatis ke jaringan WiFi’
Masukkan informasi login hanya ketika situs memiliki “https” daripada “http.” The “s” menunjukkan koneksi aman ke situs web.
Hindari masuk ke akun melalui aplikasi. Sebagai gantinya, gunakan browser internet
Selalu keluar dari akun Anda setelah menggunakannya di jaringan publik

Sebagai aturan umum, hanya terhubung ke jaringan publik ketika Anda tidak punya pilihan lain.

6. Mencegah Peretasan yang di lakukan oleh EMP

Saat memikirkan cara menghentikan peretas, sedikit yang mempertimbangkan kemungkinan serangan grid besar.

Gelombang elektromagnetik (EMP) memiliki kemampuan untuk menghancurkan perangkat listrik. EMP dapat terjadi sebagai akibat dari senjata atau fenomena alam yang digunakan.

Sangat sedikit yang dapat Anda lakukan untuk melindungi elektronik Anda dari ledakan EMP. Dianjurkan Anda mencari teknologi EMP seperti yang ditemukan di https://techprotectbag.com/.

Alat – alat ini dapat menghentikan perangkat Anda dari dihancurkan oleh EMP. Dalam serangan skala besar, alat ini bisa menyelamatkan jiwa.

7. Jangan Percayai Tautan Email

Kiat utama tentang cara mencegah peretasan adalah memeriksa semua tautan dalam surel.

Jangan pernah mengklik tautan dari email kecuali Anda memercayai sumbernya. Bahkan sumber tepercaya terkadang mengirim surel berbahaya secara tidak sengaja ketika peretasan dilakukan

Menjadi semakin umum bagi para penjahat dunia maya untuk meniru situs web yang sah. Mereka akan meminta Anda untuk masuk ke situs web perangkap ini dan akan benar-benar mencuri data Anda.

Jika ragu, ketikkan alamat situs web yang dikenal di browser web Anda daripada mengklik tautan.

8. Batasi Daftar Teman Media Sosial Anda

Memang tampaknya konyol untuk menyarankan Anda untuk membatasi anda berhubungan dengan orang asing di internet pada tahun 2019. Namun, pertimbangkan berapa banyak data pribadi yang Anda biarkan terekspos di media sosial setiap hari.

Peretas modern menggunakan teknik yang disebut rekayasa sosial. Mereka berusaha memanipulasi target dengan berteman dengan mereka. Ini sering menyebabkan mereka membiarkan informasi pribadi tergelincir secara tidak sengaja.

Rekayasa sosial juga dapat mempengaruhi Anda ketika seorang peretas menambahkan seseorang di daftar teman Anda. Mereka dapat memperoleh akses ke profil pribadi Anda dengan menjadi teman teman.

Cara terbaik untuk menghindari menjadi target rekayasa sosial adalah dengan menolak permintaan teman dari orang yang tidak Anda kenal.

9. Pertimbangkan Layanan VPN

Jika Anda ingin memaksimalkan privasi Anda, pertimbangkan untuk menggunakan layanan VPN. VPN adalah kependekan dari jaringan pribadi virtual. Jenis layanan ini melindungi alamat IP komputer Anda ketika Anda online dengan menggunakan perantara.

Alat ini melakukan semua komunikasi dengan berbagai situs web. Ini melindungi lokasi Anda, alamat komputer, data, dan riwayat penelusuran.

Biasanya, layanan ini berharga sekitar $ 5-10 per bulan. Ada juga opsi gratis, tetapi gunakan penilaian terbaik Anda saat memilih layanan.

Perlu diingat bahwa layanan VPN tidak sempurna. Lembaga penegak hukum masih dapat memaksa mereka untuk memberikan informasi Anda, jadi jangan coba gunakan untuk yang ilegal.

10. Selalu Jaga Keamanan Website Anda

Pada bulan Februari tahun 2018, lebih dari 5000 situs web diretas. Ketika penjahat dunia maya meretas situs web, mereka mencuri data seperti alamat email dan kata sandi.

Jika Anda menggunakan kata sandi yang sama pada akun email Anda, Anda dalam bahaya! Selalu ganti kata sandi Anda saat membuat akun di situs web atau blog.

Informasi pelanggan Anda juga dapat diekspos oleh peretasan.

Diperlukan untuk melakukan hal yang lebih mendetail mengenai keamanan siber untuk web jika Anda memilikinya. Langkah-langkah ini akan melindungi Anda dan pengunjung web anda
